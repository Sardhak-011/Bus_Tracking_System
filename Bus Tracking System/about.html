About SmartBus

Innovative 3-Layer Routing for Efficient Urban Transport

Project Objective

This college web project aims to modernize bus ticketing and routing with smart features: real-time booking, tracking, and adaptive routes to reduce wait times and optimize traffic. Motivation: Urban congestion and inefficient public transport in growing cities.

3-Layer Routing System
  • Fixed Layer: Pre-defined schedules for daily routes (e.g., school/office commutes).
  • Dynamic Layer: Real-time adjustments based on traffic/GPS (e.g., rerouting for jams).
  • Time-Fixed Layer: Optimized for peak hours with dedicated buses (e.g., rush hour express).
Tech Stack
  • Front-End: HTML5, CSS3 (Bootstrap 5), Vanilla JavaScript
  • Interactivity: LocalStorage for sessions, QRCode.js for tickets
  • Design: Responsive, mobile-first with custom themes
  • Deployment: GitHub Pages/Netlify (static hosting)
